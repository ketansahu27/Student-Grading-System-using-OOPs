# Student-Grading-System
In educational institutions, managing student academic records efficiently is essential. This project focuses on designing a Student Grading System using Object-Oriented Programming (OOP) principles.

The system will:
- Store student details such as name and marks
- Compute the average score
- Assign a grade based on predefined criteria

This project helps in understanding real-world applications of OOP concepts such as:
- Classes and Objects
- Encapsulation
- Methods and Functions
- Conditional Statements
